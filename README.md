Learn go programming language from [Writing Web Application](https://golang.org/doc/articles/wiki/)

## How does it work?

Run go/build with :

```
go run wiki.go

// or build
go build wiki.go
./wiki
```

Serve a website at [http://locahost:5555/view/test](http://localhost:5555/view/test)
